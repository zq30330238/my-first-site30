# 中文拟人润色技能

将机器风格中文转化为自然、有温度、像真人写的中文。

## 触发条件

用户提到以下关键词时自动调用：
- "润色"、"润一下"、"优化文字"
- "改口语化"、"更像人写的"
- "翻译感太重"、"太生硬"
- "中文优化"、"改写得更自然"

## 润色原则

### 必须做到
1. **去翻译腔**：删除不必要的"的"、"了"、"是"、"进行"、"通过"等翻译残留词
2. **口语化**：把书面语换成日常说法，比如"倘若"→"如果"，"进行购买"→"买"
3. **缩短句子**：长句拆成短句，中文更适合15-25字的松散节奏
4. **增加语气词**：适当加"吧"、"嘛"、"呢"、"啊"，但不要过度
5. **打破主谓宾僵硬结构**：中文天然松散，不需要每句都有主语
6. **成语和俗语**：合适的地方用四字词语或俗语替换绕口的表达
7. **保持专业感**：技术内容不因口语化而失去专业性

### 绝对不要
- 添加原文没有的信息
- 改变原文的事实或观点
- 过度使用网络用语（绝绝子、YYDS等）
- 把严肃内容变得轻浮
- 把"我"改成"俺"等过度口语化

## 润色层次

### 轻度（默认）
删翻译腔词汇，断长句，调语序。保留原文结构和风格。

### 中度
重新组织句子，增加自然的连接和过渡，更像中国人日常说话。

### 重度
完全重写，只保留核心信息。适合品牌文案、宣传语等需要强烈中文感的场景。

## 输出格式

```
【润色后】
（正文）

【改动说明】
- 删掉了X处翻译腔
- 重组了Y个长句
- 增加了Z处语气优化
```

## 适用场景
- 英文翻译过来的中文内容
- AI 生成的中文文档
- 品牌文案、产品介绍
- 邮件、商务沟通
- 任何需要"更像人写的"中文场景
