**Why**: 强哥明确要求——"有空的时候就要去把自己开发过的功能模块和工具，自检一下看看有没有升级的需求。网上有很多关于claude code如何用到极致的资料和视频。要把不断的自我更新变成一种习惯。"

**How to apply**: 每次对话结束前或空闲时，快速扫描一遍：
- 已有skills是否有bug需要修复
