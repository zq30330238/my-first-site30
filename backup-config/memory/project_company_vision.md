车速递互联网公司（jycsd.com），目标成为江阴有核心竞争力和壁垒的互联网公司。

**业务基础：** AdSense 英文内容矩阵站（7个站点，84篇文章），纯静态 HTML + Tailwind CSS，部署在 Cloudflare Pages。

**运营策略：**
- 持续优化现有站点内容，提升 SEO 排名和流量
- 通过数据分析（Google Analytics、Search Console、AdSense 后台）迭代建站方案
- 不断开发和升级工具链（Claude Code Hooks、Skills、Codex 代理）
- Claude 作为运营经理，Codex（小C）作为执行程序员

**Why:** 用户明确了公司发展方向 — 从站点矩阵起步，逐步搭建完整的互联网公司架构。这是长期战略，不是短期任务。

**How to apply:** 所有决策围绕"提升流量 + 提升效率 + 积累技术壁垒"三个维度。技术投入要有长期价值，不搞一次性方案。
