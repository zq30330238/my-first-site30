# Superpowers 插件

## 来源
- GitHub: `obra/superpowers`（13.5万+ Star）
- 安装: `claude plugins install superpowers-skills@anthropics-claude-code`
- 抖音博主「布鲁斯一歇」推荐

## 7 步工作流
1. Brainstorming — 动手前结构化思考
2. Git Worktrees — 分支物理隔离
3. Writing Plans — 复杂任务拆 5-10 步
4. Subagent-Driven Development — 多 Agent 并行执行
5. TDD — 测试驱动开发
6. Code Review — 5 Agent 并行审查
7. Finishing Branch — 完成收尾

## 其他推荐 Skills

- `verification-before-completion` — 完工前检查清单
