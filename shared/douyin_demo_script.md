# 抖音短视频拍摄脚本 — Claude Code 批量生成英文文章

## 基本信息
- **格式**: 9:16 竖屏 (1080×1920)
- **时长**: 25-30秒
- **风格**: 快节奏工具演示，科技感字幕
- **BGM**: 轻快电子/科技风（抖音热门BGM即可）

---

## 分镜脚本

### 镜头1: 钩子开场 (0-3秒)
**画面**: 黑底白字，快速弹出

> 字幕（居中大字）: "30篇文章，5分钟搞定？"
> 然后淡入第二行: "AI批量写文，全自动部署"

**转场**: 闪白 → 切到VSCode画面

---

### 镜头2: 一句话指令 (3-8秒)
**画面**: VSCode Claude Code 对话框，特写输入区域。键盘敲击动画（后期可叠加）

> 用户输入（画面展示）:
> "给6个网站各生成5篇新文章，然后部署上线"

**字幕（底部）**: "一句话，Claude开始干活"

**画面变化**: Claude回复"开始批量生成"的文字，快速滚动

---

### 镜头3: 加速生成过程 (8-18秒)
**画面**: 分屏或快速切换（2-3倍速）

> 画面A: 终端里文章一篇接一篇输出（article-26 OK, article-27 OK...）
> 画面B: 切到文件管理器，sub-healthy/ sub-pets/ 等目录下 .html 文件数量在增加
> 画面C: git commit + push 的终端输出快速滚动

**字幕（居中，逐条弹出）**:
> "6个子站 × 5篇 = 30篇"
> "自动排版 + SEO优化 + AdSense广告位"
> "自动提交 + 推送 + Cloudflare部署"

---

### 镜头4: 结果展示 (18-25秒)
**画面**: 浏览器打开实际网站

> 切换到 Chrome，依次打开:
> healthy.jycsd.com → 展示最新文章
> pets.jycsd.com → 展示最新文章
> finance.jycsd.com → 展示最新文章

**字幕（底部）**: "30篇英文文章，全部上线"

**画面叠加**: 绿色 ✓ 对勾动画

---

### 镜头5: 收尾CTA (25-30秒)
**画面**: 回到VSCode + Claude Code界面

> 字幕（居中大字）: "AI工具实操，关注我看更多"
> 底部小字: "Claude Code + DeepSeek | 全自动内容矩阵"

**画面角落**: 头像/抖音号 JYSMJS 露出

---

## 拍摄清单

| # | 素材 | 录制方式 |
|---|------|---------|
| 1 | VSCode输入指令 | 屏幕录制（区域裁剪为9:16） |
| 2 | 终端生成过程 | 屏幕录制（可倍速） |
| 3 | 文件目录变化 | 屏幕录制文件管理器 |
| 4 | 浏览器验证 | 屏幕录制Chrome |
| 5 | 字幕叠加 | 剪映/CapCut后期 |

## 录制步骤（实操）

1. **打开OBS或Windows自带录屏**，设置9:16竖屏区域
2. **在Claude Code中输入**: "给6个网站各生成1篇新文章，部署上线"
3. **录制完整过程**，包含输入→生成→部署→浏览器验证
4. **导入剪映**，加速中段（2-3x），加字幕和BGM
5. **导出** 1080×1920，发布抖音

## 后期要点
- 字幕字体：推荐「优设标题黑」或抖音热门科技字体
- 关键数字（30篇/5分钟/6个站）用放大+高亮动画
- BGM节奏对齐画面切换点
- 开头3秒必须有钩子，否则划走率太高
