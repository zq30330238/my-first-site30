## CTR与流量增长策略

### 标题优化
- "How to Learn Any Language in 3 Months: The Immersion Method"
- "7 Study Techniques Backed by Neuroscience (2026 Guide)"
- 数字+方法论+年份
- 问题式标题占比30%

### Meta Description
- 具体收益 + CTA
- 例: "Master any subject with 7 neuroscience-backed study techniques. Learn spaced repetition, active recall, and the Feynman method. Boost your exam scores today."

### Featured Snippet
- 学习步骤有序列表
- "What is spaced repetition?" → 40-60字定义
- 方法论对比用表格

### 内链策略
- 学习技巧↔记忆专注↔考试准备三角互链
- 每篇文章引用相关方法论的其他文章
- Related Articles + Read Next

### 内容新鲜度
- 学习平台对比标注 "Compared: June 2026"
- 考试信息按考试季更新
- 首页 "Most Popular Study Guides" 区块

### 回访机制
- 学习计划系列: "30-Day Study Transformation"
- 打印友好: 学习计划表+记忆卡片模板可打印

### 富文本摘要
- FAQ Schema
- HowTo Schema (学习步骤文章)
- BreadcrumbList Schema

## 质量与UX标准

### 内容质量
- 每篇文章 ≥1200字，学习技巧类含具体可操作步骤/时间表
- 文章结构: 导语钩子 → ToC → Key Takeaways → 正文3段 → 学习计划模板 → FAQ → 结论
- 阅读时间标注 + 最后更新日期
- 禁止AI cliché
- 引用真实学习法来源(Active Recall/Ebbinghaus/Feynman等有据可查的方法)

### 视觉质量
- 封面图16:9，lazy loading，alt文本
- 紫色`#7C3AED`在白色底上对比度通过WCAG AA
- 学习计划表: 斑马条纹，语义化`<table>`
- 禁止图片阴影遮罩

### UX设计
- 面包屑 + 文章ToC(桌面sticky) + 相关文章(3篇) + 返回顶部 + 阅读进度条
- 移动端纯CSS汉堡菜单
- **学习技巧卡片**: step-by-step数字编号，CSS counter实现
- 记忆技巧: 信息框(Info Box)，浅色背景+左边框

### SEO
- Schema: Article + BreadcrumbList + Organization + WebSite + FAQ
- Meta/OG/Twitter/Canonical 完整
- 内部链接每篇2-3个

### 广告位
- 3个/篇，纯自动广告，禁止手动广告位

### 性能 + 无障碍
- 预连接 + lazy loading + 零JS + 语义化HTML + Skip to content + 键盘可导航
